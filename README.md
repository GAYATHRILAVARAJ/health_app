This is a healthcare app which has the following functionalities:
- BMI Calculator
- Appointment Booking
- Viewing appointments
- Diet suggestion
- Medicine Reminder
- Symptom Checker
- Assistant chatbot
- To run the program in VS code:
-  In terminal:
-               1.venv\Scripts\activate
-               2.python health_app.py
-  pip installation of wxPython must also be done prior to the execution of code.
